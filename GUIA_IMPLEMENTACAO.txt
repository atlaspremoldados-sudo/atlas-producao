## 📋 FORMULÁRIOS DISPONÍVEIS

1. **Máquina 01** (Blocos/Canaletas)
   - Registra produção em tempo real
   - 9 produtos diferentes
   - Rastreia insumos

2. **Máquina 02** (Pisos Dormido)
   - Registra produção em tempo real
   - Qualidade do concreto (MPA)
   - 9 produtos diferentes

3. **Máquina 03** (Mesa Vibratória)
   - Complementos (Guias, Elementos, Concregrama)
   - Rápido e simples

4. **Desforma + Paletização Máq02**
   - Linkado automaticamente à produção anterior
   - Registra 1ª linha, 2ª linha, rejeição
   - Identifica defeitos

5. **Paletização Máq01**
   - Linkado automaticamente à produção anterior
   - Rastreia paletes (para recuperação)
   - Cálculo automático de rejeição

6. **Entregas**
   - Múltiplos clientes (até 3)
   - Km rodado + Tempo parado (Munck)
   - Rastreamento de paletes
   - Integração com Forteplus (estoque/preços)

7. **Funcionários**
   - Registro de presença
   - Observação individual por funcionário
   - Anotações do dia

## 🚀 PRÓXIMOS PASSOS (HOJE)

### PASSO 1: Criar Conta Supabase (5 min)
└─ Acesse: supabase.com
└─ Clique "Start your project"
└─ Use email: joaorissibr@gmail.com
└─ Crie projeto: "atlas-producao"
└─ Aguarde 2-3 minutos

### PASSO 2: Pegar Credenciais (2 min)
└─ Settings → API
└─ Copie: Project URL e anon public key
└─ Guarde em lugar seguro

### PASSO 3: Criar Tabelas (5 min)
└─ No Supabase, vá para: SQL Editor
└─ Cole o código SQL do arquivo: supabase-schema.sql
└─ Clique "Run"
└─ Aguarde conclusão

### PASSO 4: Clonar/Deploy (10 min)
OPÇÃO A - Localmente:
  $ git clone [seu-repositório]
  $ cd atlas-producao
  $ npm install
  $ npm run dev
  → Acessa: http://localhost:5173

OPÇÃO B - Em Produção (Vercel):
  $ npm run deploy
  → Recebe um link público
